# snapcd_module_param_from_definition (Data Source)

Use this data source to access information about an existing Module Param (From Definition) in Snap CD.


## Example Usage

```terraform
data "snapcd_stack" "default" {
  name = "default"
}

data "snapcd_namespace" "mynamespace" {
  name     = "mynamespace"
  stack_id = data.snapcd_stack.default.id
}
data "snapcd_module" "mymodule" {
  name         = "mymodule"
  namespace_id = data.snapcd_namespace.mynamespace.id
}
data "snapcd_module_param_from_definition" "myvar" {
  name      = "myvar"
  module_id = data.snapcd_module.mymodule.id
}
```
