# snapcd_integration_stack_supply (Resource)

Manages an Integration Stack Supply in Snap CD. Supplies the integration to every module under the given stack.


## Example Usage

```terraform
resource "snapcd_integration_stack_supply" "alerts" {
  integration_id = data.snapcd_integration.alerts.id
  stack_id       = snapcd_stack.example.id
}
```
