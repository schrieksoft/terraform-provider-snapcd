# snapcd_integration_namespace_supply (Resource)

Manages an Integration Namespace Supply in Snap CD. Supplies the integration to every module in the given namespace.


## Example Usage

```terraform
resource "snapcd_integration_namespace_supply" "alerts" {
  integration_id = data.snapcd_integration.alerts.id
  namespace_id   = snapcd_namespace.example.id
}
```
