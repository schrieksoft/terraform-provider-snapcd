# snapcd_namespace_integration_event (Resource)

Manages a namespace-scoped Integration Event in Snap CD.


## Example Usage

```terraform
resource "snapcd_namespace_integration_event" "notify_on_failure" {
  integration_id = data.snapcd_integration.alerts.id
  namespace_id   = snapcd_namespace.example.id
  trigger        = "JobFailed"
  template       = "Job failed on namespace: {{jobName}}"
}
```

## Schema

### Required

- `integration_id` (String) ID of the target integration.
- `namespace_id` (String) ID of the Namespace this event is scoped to.
- `trigger` (String) Trigger this subscription fires on. Must be one of 'JobSucceeded', 'JobFailed', 'JobAwaitingApproval', 'JobApproved', 'JobDeclined', 'JobCancelled', 'MissionStarted', 'MissionMilestoneReported', 'MissionCompleted', 'MissionFaulted'.

### Optional

- `filter` (String) Optional filter expression.
- `is_disabled` (Boolean) Whether the subscription is disabled.
- `template` (String) Optional message template ({{token}} substitution). Omit to use the default for the trigger.

### Read-Only

- `id` (String) Unique ID of the Integration Event.
